# Customer Retention Intelligence

### Churn Prediction & Retention Analytics Platform

An early warning system that predicts customer churn using machine learning and explains the key drivers behind attrition.
Designed to support proactive, data-driven customer retention decisions.

---

## Key Features

* Churn probability prediction
* Tuned decision threshold for business use
* Explainable insights using SHAP
* Interactive Streamlit dashboard
* Downloadable customer retention report

---

## Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn, XGBoost
* SHAP
* Streamlit

---

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
